Never abbreviate variable names. Always use full, descriptive names.

**Why:** User preference for readability and consistency.

**How to apply:** No single-letter variables (e.g. `c`, `e`, `i` unless a standard loop index), no truncated names (e.g. `req` → keep as `req` is idiomatic, but `c` for `cents` is not OK). When in doubt, spell it out.
